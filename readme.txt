== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'tweetrly-auto-tweets' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "Dashboard" > "Auto-Tweets" and fill out the options there.
